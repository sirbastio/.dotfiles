# GitHub Dark Default for Yazi

A local Yazi flavor matched to this dotfiles repository's existing themes:

- Ghostty's bundled `GitHub Dark Default` palette supplies the terminal, ANSI,
  cursor, and selection colors.
- `projekt0n/github-nvim-theme`'s `github_dark_default` palette supplies the
  canvas, border, semantic, diagnostic, and syntax-highlighting colors.

The flavor intentionally does not set `[app].overall.bg`, so Ghostty's
configured background opacity and blur continue to work.
